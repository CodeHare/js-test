# 1.2.2 Jest  测试框架


## install

```
npm install --save-dev jest

# package.json：

{
  "scripts": {
    "test": "jest"
  }
}
```

## demo

```
#index.js

function sum(a, b) {
  return a + b;
}
module.exports = sum;


#index.test.js
const sum = require('./index');

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3);
});
```
- `npm run test`
- 测试后 Jest 将打印测试信息
